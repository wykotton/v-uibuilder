# 事件
事件配置是建立组件之间数据交互和动作的纽带。

## 事件消息
点选你需要配置的组件，在右侧属性配置窗体中对应的消息面板下有该组件拥有的所有消息事件
![](/images/event-handler/0.png)


点击展开需要配置的事件，点击前面的图标即可编辑自定义事件消息。
![](/images/event-handler/1.png)

## 路由配置
作为UIBuilder中最为重要的元件之一，路由配置器是组件之间从发起到接收建立联系的重要通道，如下图所示。
